## 作成した画面一覧（7画面）

### 1. トップページ（メインダッシュボード）
**ファイル**: `screen_01_top_page.html`
- 主要機能へのアクセス
- バス停検索機能
- クイックアクセス（時刻表、路線検索、運行状況、お知らせ）
- 重要なお知らせの表示

### 2. バス停検索・時刻表画面
**ファイル**: `screen_02_bus_stop_timetable.html`
- バス停名・路線名検索
- 現在地から最寄りバス停検索
- バス停詳細情報（住所、電話番号、車椅子対応等）
- 路線別時刻表表示（平日・土曜・日曜）
- リアルタイム到着予測

### 3. 路線検索・ルート案内画面
**ファイル**: `screen_03_route_search.html`
- 出発地・目的地入力
- 検索オプション（時刻指定、乗り換え回数制限）
- 複数ルートの比較表示
- ルート詳細（所要時間、運賃、乗り換え回数、徒歩時間）
- ステップバイステップの案内
- お気に入り登録・共有機能

### 4. リアルタイム運行状況画面
**ファイル**: `screen_04_realtime_status.html`
- 運行状況概要（正常・遅延・運休の統計）
- リアルタイム地図表示
- 路線別運行状況一覧
- 遅延情報の詳細表示
- フィルター機能（遅延のみ、正常のみ等）

### 5. お知らせ・運行情報画面
**ファイル**: `screen_05_news_announcements.html`
- カテゴリ別お知らせ（運行情報、重要なお知らせ、イベント情報等）
- 検索・フィルター機能
- 重要度別の表示（重要・遅延・情報・イベント）
- 通知設定の管理
- 詳細表示・共有機能

### 6. マイページ・設定画面
**ファイル**: `screen_06_mypage_settings.html`
- ユーザープロフィール表示・編集
- お気に入り管理（バス停・路線・ルート）
- 通知設定（プッシュ・メール・SMS）
- 表示設定（テーマ・フォントサイズ・言語）
- 位置情報・プライバシー設定
- アカウント管理機能

### 7. ヘルプ・お問い合わせ画面
**ファイル**: `screen_07_help_contact.html`
- FAQ（よくある質問）
- 利用ガイド（機能別説明）

- 連絡先情報
- フィードバック機能

## アクセシビリティ対応

### 視覚的アクセシビリティ
- **コントラスト比**: WCAG AA準拠（4.5:1以上）
- **色覚異常対応**: 色だけでなくアイコンやパターンでも情報を伝達
- **フォントサイズ**: ユーザー設定に応じた調整
- **ダークモード**: システム設定に連動

### 操作のアクセシビリティ
- **タッチターゲット**: 最小44px × 44px（タッチフレンドリー）
- **キーボードナビゲーション**: 全機能がキーボードで操作可能
- **フォーカス表示**: 明確なフォーカスインジケーター
- **スクリーンリーダー**: 適切なHTML構造と代替テキスト

### 認知アクセシビリティ
- **アニメーション制御**: ユーザーの設定に応じた制御
- **エラーハンドリング**: 明確なエラーメッセージ
- **ローディング状態**: 適切なフィードバック
